# PhoneBio Agent Instructions

PhoneBio builds an offline-first voice support agent for camera-free field biology work.

## Boundaries

- Do not require camera input for v1 workflows.
- Prefer local, versioned knowledge over live web lookup during calls.
- Never store API keys, test phone numbers, or private caller data in git.
- Treat safety guidance as protocol lookup and triage support, not medical or regulatory certification.
